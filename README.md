# mcpcli

CLI for interacting with MCP servers. TypeScript alternative to [mcptools](https://github.com/f/mcptools).

## Usage

```bash
npx mcpcli tools node ./my-server.js
npx mcpcli call my-tool node ./my-server.js -p '{"key": "value"}'
npx mcpcli resources node ./my-server.js
npx mcpcli prompts node ./my-server.js
```

## Install

```bash
npm install -g mcpcli
```
